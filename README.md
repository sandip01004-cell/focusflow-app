# 🧠 FocusFlow
### Pomodoro · Tasks · Study Analytics

FocusFlow is a minimal, modern productivity web app that combines a **Pomodoro timer**, **to-do list**, and **study analytics** into one distraction-free interface.

It runs entirely in the browser with no backend required.

---

## 🚀 Features

### ⏱️ Pomodoro Timer
- Custom study time, break time, and cycles
- Auto mode for continuous sessions
- Fullscreen distraction-free timer
- Tap anywhere to pause (no buttons)
- Timestamp-based timer (works after sleep/inactivity)

---

### ✅ To-Do List
- Create tasks with title and description
- Use `#tags` for categorization
- Priority levels (Low, Medium, High, Urgent)
- Link tasks to Pomodoro sessions
- Filter by tags and priority

---

### 📊 Study Analytics
- Track:
  - Today's study time
  - Weekly study time
  - All-time stats
- Session history (bank-statement style)
- Tag-based analytics (e.g. `#physics`)
- Time spent per subject/tag

---

### ⚙️ Settings
- Dark / Light mode
- Liquid glass UI toggle
- Sound presets
- Notifications
- Reset data options

---

## 🎨 UI / UX

- Minimal, Notion-inspired design
- Smooth page transitions
- Scroll-based navigation between pages
- Fullscreen timer animation
- Eye-friendly dim grey timer

---

## 🧠 How It Works

- Fully client-side application
- No backend required
- Works offline
- Uses browser storage for persistence

### 💾 Data Stored:
- Tasks
- Session history
- Timer state
- Settings

---

## 🌐 Deployment (GitHub Pages)

1. Push project to GitHub
2. Go to **Settings → Pages**
3. Select branch (`main`)
4. Deploy

---

## ⚠️ Notes

- Data is stored per browser
- Clearing browser data will erase progress
- No cross-device sync
